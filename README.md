# Portfolio Source Code

Welcome to the repository for my personal portfolio. Yes, you are looking at the code for the site you probably just clicked away from.

## 🤔 The "Why"
If a developer builds a portfolio and doesn't put it on GitHub, does the CSS even render? This repo exists because:

* **Version Control**: Because "index_final_v2_REALLY_FINAL.html" is not a strategy.
* **Proof of Life**: It proves I actually spent time making this and didn't just find it in a digital lost-and-found.
* **Workflow**: Automated deployments via Vercel, because manual uploads are for 2005.

## 🛠️ Tech Stack
* **HTML5/CSS3**: Custom layouts using CSS Variables and Flexbox—no heavy frameworks.
* **Vanilla JavaScript**: Powers the custom interactive cursor and scroll-reveal animations.
* **SVG Power**: High-contrast tech logos to ensure the skills section actually pops.

## ✨ Highlights
* **Dark Mode by Default**: Because our eyes deserve a break.
* **Responsive Layout**: Engineered to look sharp on everything from a massive desktop to your phone.
* **Vercel Integration**: Built for instant previews and CI/CD pipelines.
